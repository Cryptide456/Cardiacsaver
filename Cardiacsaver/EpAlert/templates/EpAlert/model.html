How the machine learning model works

One of the hardest parts of designing this project was creating the machine learning algorithm and then integrating into the website, especially because we had minimal experience in the past. To get the machine learning to work properly, we used a “train set” and a “test set”. Much like teaching a student math, the train set would contain both the problem and the solution that we want the machine to learn. The test set would only contain the problem in order to test the algorithm on whether or not it was correct. After running the train and test set on nine models, we got the accuracy of each. For our data, the most reliable models were the AdaBoost and Gradient Boost classifiers with around a 98% accuracy rate. Therefore, to get the most accurate result these two, AdaBoost and Gradient, Boost take precedence over other models to show whether a person will have a heart attack or not. To create a better experience for our user, we made our AI explainable (AI shows the steps it took to run) by displaying the output of each method and then telling the user the process behind our conclusion.
